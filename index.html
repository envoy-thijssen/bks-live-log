Use case

Wat dit oplost

Je bent op een druk festival en wilt later nog weten: wie heb ik gezien, wanneer, op welk podium, wat vond ik ervan, en hoe verhoudt dat zich tot andere jaren?

Workflow

Telegram → JSON → dashboard

Ratings kunnen eerst via Telegram worden ingevoerd. Daarna valideert Hermes artiest, tijdslot en podium, en schrijft alles weg als nette records die dit dashboard leest.
